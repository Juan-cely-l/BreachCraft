# BreachCraft: Romper para entender, construir para asegurar.
Mi laboratorio de seguridad ofensiva enfocado en el compromiso de activos en entornos controlados (TryHackMe/HTB). Un punto de intersección entre automatización en Python, persistencia y documentación técnica de alto nivel. Encuentra el camino de entrada para construir la defensa.
